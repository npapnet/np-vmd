Introduction
============

Welcome to the documentation for **np_vmd**, a comprehensive Python package for the analysis of vibrational systems. 
This package provides tools for simulating and analyzing single-degree-of-freedom (SDOF), two-degree-of-freedom (2DOF),
and multi-degree-of-freedom (multi-DOF) systems.
